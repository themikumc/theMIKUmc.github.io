# TheMikuMC Portfolio

Minimal Vue + Vite + Tailwind portfolio with Nginx deploy.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Docker (Nginx)

```bash
docker compose up --build
```

## Reviews

Edit `src/data/reviews.js` and add a new object:

```js
{
  id: 4,
  name: 'Client Name',
  role: 'Server Owner',
  quote: 'Review text.'
}
```
